# leetcode_visualizer"

## Create and start virtual env:
`python3 -m venv .venv`

`source .venv/bin/activate`

## pytest
### install pytest
`pip install pytest`

### running tests
use `pytest` to run tests, or `pytest -k {test_name}` to run a specific test


## Jupyter:
### Install Jupyter
`pip install notebook`

### Launch Jupyter:
`jupyter notebook`

browser should open to http://localhost:8888/tree
